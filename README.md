# Codex Skills

Personal Codex skills for code review, database review, and Taiwan stock snapshots.

## Skills

- `code-reviewer`: Review diffs, pull requests, branches, or pasted code for correctness, security, performance, and maintainability.
- `sql-review`: Review SQL, schemas, indexes, query plans, migrations, ORM queries, and time-series workloads across SQL Server, MySQL, Oracle, PostgreSQL, and InfluxDB.
- `realtime-tw-stock-snapshot`: Fetch on-demand Taiwan stock quotes, institutional flows, historical volume structure, price-band volume, and USD/TWD context.

## Installation

Copy a skill directory into your Codex skills directory, or reference it from a Codex plugin. Each directory is self-contained and includes its own `SKILL.md`.
